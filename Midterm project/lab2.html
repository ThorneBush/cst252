Task 2: Using a Text Editor
Atom is a text editor, a program that edits plain text files (unlike a word processor like Word) and is free. You should already have installed it. It is okay if you use Sublime3, Brackets, NotePad++, TextEdit, Vim, Emacs, or another text editor, but I won't be able to support you if you have problems.

You can customize Atom and add plugins. It is very powerful. If there is something you'd like it to do, google to see if someone can guide you.
